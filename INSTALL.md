# 🛠️ Installation & Update Guide

Welcome to the Local Handouts Manager! This guide covers how to get the application running on your machine and how to safely update it later without losing your campaign data.

## Prerequisites

* **Python 3.8+** installed on your system.
* **Git** (recommended for easy updates).

---

## First-Time Installation

### 1. Get the Code
Clone the repository to your local machine:
```bash
git clone [https://github.com/YOUR_USERNAME/local-handouts-manager.git](https://github.com/YOUR_USERNAME/local-handouts-manager.git)
cd local-handouts-manager
